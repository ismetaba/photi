# T11 Test Plan — Organizer Pages

## Acceptance Criteria
1. `Home` lists owned events as cards (title, status chip, cover) with Yeni Etkinlik CTA via the layout.
2. `NewEvent` form fields: title, startsAt, endsAt, brandingColor, optional brandingLogoUrl. Submits via `api.post('/events', ...)`; on success navigates to `/events/:id`.
3. `EventDetail` shows tab navigation (Fotoğraflar, Katılımcılar, Ayarlar, QR) with placeholder implementations.
   - **PhotosTab**: Dropzone uploads (parallel ≤ 4, queues retries on failure), photo grid uses lazy thumbnails, each card shows matchCount badge + Featured toggle, clicking opens Lightbox modal that lists prev/next.
   - **ParticipantsTab**: list with selfie thumb + match count.
   - **SettingsTab**: form to edit title/dates/brand color, Publish button (disabled when no ready photos), Archive button.
   - **QrTab**: shows QR PNG image + share link with Copy button + Download link.
4. `Billing` shows balance card, three packages with Buy button → modal with mock card form → submitting calls `api.post('/billing/purchase')` and refetches `me`.
5. LowCreditBanner already shows globally via OrganizerLayout (T10).

## Test Scenarios (Vitest)
- **Home**: with `useMyEvents` mocked, renders three cards with their titles + status. Empty state visible when list is empty.
- **NewEvent**: filling the form and submitting calls `api.post('/events', {...})` once with the right body and navigates to the returned event id.
- **PhotosTab Dropzone**: dropping files invokes the upload mutation once per file (mocked), shows uploading → done states, retries failed uploads (we'll mock a failure).
- **Lightbox**: clicking a photo opens the modal; pressing Escape closes; arrow keys navigate.
- **SettingsTab**: Publish button is disabled when there are no ready photos; enabled when at least one ready photo exists. Clicking Publish calls publish endpoint.
- **Billing**: rendering with `me` mocked shows balance; clicking a package opens modal; submitting modal calls purchase mutation; balance refetched.
