# T01 Test Plan — Bootstrap Monorepo

## Acceptance Criteria
1. pnpm workspace at root recognises `apps/*` and `packages/*`.
2. `tsconfig.base.json` exists at the repo root with shared compiler options.
3. `packages/shared` exports `src/index.ts` (placeholder is fine for T01).
4. `apps/web` is a Vite + React 18 + TS scaffold with Tailwind configured (brand colors `#0F1B3D`, `#FF6A1A`, white bg) and Vitest with jsdom setup.
5. `apps/backend` is a Fastify + better-sqlite3 + Drizzle scaffold with `tsx` dev script, `multipart`, and `cors` plugins registered.
6. Root scripts: `dev` runs backend and web in parallel; backend exposes `db:migrate`.
7. Top-level `README.md` documents 3-command setup.
8. Root `pnpm test --run` finishes cleanly (no failing tests).

## Test Scenarios
- **Layout check**: Vitest test asserts presence of root `package.json`, `pnpm-workspace.yaml`, `tsconfig.base.json`, `apps/web/package.json`, `apps/backend/package.json`, `packages/shared/package.json`, `README.md`.
- **Workspace config**: Test parses `pnpm-workspace.yaml` and asserts it lists both `apps/*` and `packages/*`.
- **Tailwind config**: Test reads `apps/web/tailwind.config.ts` and asserts it includes the brand colors `#0F1B3D` and `#FF6A1A`.
- **Web package.json**: Test asserts `vite`, `react`, `tailwindcss`, `vitest`, `jsdom` are in deps/devDeps; `test` script exists.
- **Backend package.json**: Test asserts `fastify`, `@fastify/multipart`, `@fastify/cors`, `better-sqlite3`, `drizzle-orm`, `tsx` are listed; scripts include `dev` and `db:migrate`.
- **Shared package**: Test asserts `packages/shared/src/index.ts` exists.
- **Root scripts**: Test asserts root `package.json` has a `dev` script that references both apps and a `test` script.
